sync.tvguitar.com lets users browse public sync songs and, when logged in, publish or update songs they own. This notice explains what information is used when you visit or use the service.

What data is used

Supabase handles email and password authentication. The app server stores song sync data, user profile defaults, and publishing metadata in PostgreSQL.

Profile defaults

Shortcut mappings, auto-fullscreen behavior, helper visibility, and other non-song defaults are stored in the authenticated user profile so they can follow the user across sessions.

Published songs

Published songs can remain visible after account deletion or closure so the public practice library can stay available. If you publish a song, you should only include material you have the right to share.

Analytics and consent

The site uses Google Tag Manager to load Google Analytics only after you accept optional cookies. If you choose essential-only storage, analytics tags do not load. When enabled, analytics may collect device, browser, page view, and interaction data to help measure site usage and improve the service.

Cookies and local storage

Essential browser storage is used for sessions, local draft recovery, theme preference, and consent choices. Optional analytics storage is only enabled after consent.

Your rights

You can request account deletion, ask questions about your data, or contact the site owner using the email above. You can also decline optional analytics by choosing essential-only cookies in the consent banner.
